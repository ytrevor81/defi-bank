# defi-bank
A decentralized bank on Rinkeby, where you can deposit ETH and withdraw your ETH + interest, in the form of Trev Bank Currency (TBC)

If you have a Rinkeby account connected to your MetaMask wallet, check out my DeFi app here: https://ytrevor83.github.io/defi-bank-live/

## Technologies
Solidity (Smart Contracts)  |  React  |  Javascript  |  Web3.js  |  Truffle  |  Ganache  |  Infura API
